# Uzdevumu pārvaldības sistēma

## Apraksts
Vienkārša uzdevumu pārvaldības sistēma Python valodā ar CLI un Git darba plūsmu.

## Palaišana
python src/main.py
`
## Funkcionalitāte

* Uzdevumu pievienošana
* Dzēšana
* Statusa maiņa
* Saglabāšana un ielāde no faila
* Vienību testi

## Statusi

* Jauns
* Procesā
* Pabeigts
* Atlikts

## Zaru stratēģija

* main – stabila versija
* dev – izstrāde
* feature/* – funkcionalitātes
* fix/* – kļūdu labojumi

* Maintainer
* Testētājs
